Sie können auf der Einstellungsseite konfigurieren, welche Funktionen aktiviert werden sollen.

✨ Features

Auf Netflix, Prime Video, Disney+ & Hotstar und Crunchyroll wird automatisch:
• Intros & Zusammenfassungen übersprungen
• Entweder Abspann übersprungen
• Oder Abspann immer angezeigt
• Werbung übersprungen
• Zeigt TMDB-Bewertung
• Geschwindigkeitsregler hinzugefügt
• Verbessert das Aussehen von Untertiteln
• Abgespielt, wenn Vollbildschirm geöffnet wird

Zusätzlich zu den geteilten Funktionen:

Auf Netflix wird auch automatisch:
• Automatisch zuletzt verwendetes Profil ausgewählt
• Inaktivitätswarnung übersprungen („Schauen Sie weiter?“)

Auf Amazon Prime-Video wird auch automatisch:
• Eigenwerbung übersprungen
• Kostenpflichtige Inhalte gefiltert
• Die lästige Unschärfe, wenn das overlay angezeigt wird entfernt

Auf Crunchyroll wird auch automatisch:
• Filter zum Release Kalender hinzugefügt

Android: 
Prime Video und Disney+ (im Desktop-Modus) werden auf Android unterstützt. Netflix funktioniert nicht auf Android.

Zur Verwendung auf einem Android Handy Laden Sie 
[Kiwi Browser] (https://play.google.com/store/apps/details?id=com.kiwibrowser.browser) 
oder eine ähnliche App herunter, um das Chrome-Addon installieren zu können.
